# Imagens do site

Coloque aqui as imagens referenciadas no `src/app/config.json`. Os caminhos no config são relativos a esta pasta (ex.: `hero.jpg` = `public/images/hero.jpg`).

## Estrutura esperada

| Caminho no config | Descrição |
|------------------|-----------|
| `images/hero.jpg` | Imagem de fundo da seção Hero |
| `images/about/coach.jpg` | Foto do treinador (seção Sobre) |
| `images/local.jpg` | Imagem da seção Localização |
| `images/conquistas/copa-cisma.jpg` | Conquista Copa CISMA 2025 |
| `images/conquistas/taca-ouro.jpg` | Conquista Taça Ouro |
| `images/conquistas/treinos.jpg` | Conquista Treinos Intensivos |
| `images/conquistas/uniao.jpg` | Conquista União e Dedicação |

Você pode usar `.jpg`, `.jpeg`, `.png` ou `.webp` — basta ajustar a extensão no `config.json`.

**URLs externas:** no config, se você colocar um link começando com `http://` ou `https://`, ele será usado como está (sem passar pela pasta public).
